# KOALA_FRONT_END

---

This project is KOALA (provide chatting service and keyword alert)

### 프로젝트 가져오기

```
git clone -
```

### 배포 명령어

```
npm run build
# if you using yarn
yarn build
```

### 개발 명령어

```
npm start
# if you using yarn
yarn start
```

### 테스팅 명령어

```
npm run test
# if you using yarn
yarn test
```
